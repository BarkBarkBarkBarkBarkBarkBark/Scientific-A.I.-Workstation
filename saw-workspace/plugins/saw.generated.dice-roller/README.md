# saw.generated.dice-roller

This plugin simulates rolling dice.

## Files

- `plugin.yaml`: manifest for the dice roller plugin.
- `wrapper.py`: implements the logic for rolling dice.

## Quick Usage

- Define the number of sides of the dice.
- Define the number of dice to roll.

## Runtime Contract

- Inputs: number of dice, number of sides.
- Outputs: the results of rolling the dice.
